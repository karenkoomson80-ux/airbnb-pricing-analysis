# Airbnb Pricing Analysis – Los Angeles

## Overview
This project analyzes 45,000+ Airbnb listings in Los Angeles to identify factors that influence rental pricing.

## Objective
To explore which variables drive differences in Airbnb prices and generate insights for a real estate consulting scenario.

## Tools Used
- R
- dplyr
- ggplot2

## Data Source
Inside Airbnb – Los Angeles dataset (September 2024 snapshot)

## Files Included
- R analysis script
- Final project report
